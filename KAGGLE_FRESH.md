# Kaggle clean training

1. Upload `absa_clean_ablation_fresh.zip` as a Kaggle Dataset.
2. Create/open a Kaggle Notebook, add that Dataset as Input.
3. Turn on GPU and Internet.
4. Upload/open `kaggle_ablation_fresh.ipynb` or use the copy included inside the zip.
5. Run all cells.

The notebook extracts source code into `/kaggle/working/absa_clean_ablation_fresh`, removes old output folders, and trains from scratch.
